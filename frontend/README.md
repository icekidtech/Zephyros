# Zephyros II - Blockchain Supply Chain Verification

Zephyros II is a Next.js application that leverages blockchain technology to verify and track products throughout the supply chain, ensuring authenticity and transparency.

## Features

- Product tracking and verification using blockchain technology
- QR code generation for product verification
- Dashboard for monitoring supply chain activities
- User authentication and authorization
- Responsive design for all devices

## Tech Stack

- **Frontend**: Next.js, React, TypeScript, Tailwind CSS
- **UI Components**: Radix UI, shadcn/ui
- **Blockchain**: Ethereum (via ethers.js)
- **Authentication**: NextAuth.js
- **Styling**: Tailwind CSS
